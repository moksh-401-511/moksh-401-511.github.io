Calibration of Object Recognition Models

Trained models on CIFAR-100 dataset using temperature scaling, data augmentation, and class-aware thresholding. Fine-tuned decision layers post-training to lower validation loss.

Hand-Pose Recognition for Robot Control

Implemented Histogram of Oriented Gradients (HOG) from scratch for feature extraction of hand images. Trained Support Vector Machine (SVM) on HOG features to detect open or closed hand gestures. Integrated hand-recognition pipeline to control TurtleBot robot in simulation.

Panoramas Creation with Image Stitching

Implemented SIFT feature detection and matching from scratch. Utilized RANSAC algorithm based on SIFT matched features to find homography matrix between images. Applied backward warping and Laplacian blending for seamless image stitching and panorama generation.
